# MoodFlow · 发展路线（Roadmap）

本文概述项目未来版本的规划与优先级。路线会随社区反馈与实际使用动态调整。

## 近期（v1.x）以用户体验为核心
- v1.1 统计与检索增强
  - 趋势图交互优化：缩放/平移、区间选择
  - 占比图点击联动至搜索页（自动带入表情过滤）
  - 搜索页导出结果 JSON、更多过滤（仅有备注/仅有心情/空心情）
- v1.2 数据导入导出与备份
  - 云备份：一键上传 JSON 到云存储并生成下载链接
  - 导入向导与校验（预览差异、按 ts 合并策略说明）
- v1.3 主题与外观
  - 自定义主题色、表情样式与尺寸
  - 无障碍适配（对比度、字号、可读性）
- v1.4 本地数据安全
  - 口令加密本地存储（AES），口令找回提示
  - 风险提示与性能评估（加密后的读写影响）
- v1.5 多语言（i18n）
  - 中/英切换；文案抽取与国际化资源

## 中期（v2.x）以可扩展性为核心
- v2.0 插件化与数据模型扩展
  - 心情记录扩展字段（标签、图片、评分）
  - 插件扩展点（自定义统计、导出器、可视化）
- v2.1 同步策略与冲突解决
  - 替换/升级“后写优先”为可配置策略（如“手动合并/保本地/保云端”）
  - 冲突日志与可视化比对
- v2.2 跨端与生态
  - Web/H5 轻应用（Taro/UniApp 评估）
  - 数据格式标准化，便于多端共享

## 工程与质量（持续推进）
- 测试
  - 单元测试（工具方法、storage/sync）
  - UI 自动化冒烟（关键流程：记录、备注、导入、同步）
- CI/CD
  - Lint/格式化检查与自动化测试
  - 预览包构建（体验版）
- 性能
  - 大月/大数据量渲染优化
  - 同步/导入导出耗时可视化与合理提示
- 安全与隐私
  - 最小化权限、敏感信息隔离
  - 完善隐私政策与数据生命周期说明

## 社区与治理
- 贡献指南与受理流程（Issues/PR 模板）
- 版本语义化（SemVer）与里程碑规划
- 定期发布节奏与发布说明（CHANGELOG）

## 里程碑建议（弹性）
- 2025-Q4：v1.1/v1.2（统计检索增强、云备份导出）
- 2026-Q1：v1.3/v1.4（主题外观、本地加密）
- 2026-Q2：v1.5 & v2.0（i18n、插件化雏形）

欢迎在 Issues 中提出建议与需求，共建 MoodFlow！
